# roadmap: arg passthrough

> **root motto:** don't let declastruct interfere with --arg passthrough

---

## overview

execute the blueprint in 4 phases:
1. test fixture — create wish-with-args.fixture.ts
2. plan command — add passthroughArgs param and process.argv injection
3. invoke command — add help text and unknown option guidance
4. documentation — update readme.md
