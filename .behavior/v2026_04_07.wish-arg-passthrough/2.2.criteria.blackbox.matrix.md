# blackbox coverage matrix: arg passthrough

---

## matrix.1: plan command arg passthrough

| ind: separator | ind: args count | dep: process.argv contains | dep: plan uses |
|----------------|-----------------|---------------------------|----------------|
| none           | 0               | empty (node + entry only) | defaults       |
| none           | 1 (`--env prod`) | `['--env', 'prod']`      | prod config    |
| none           | 2+ (`--env prod --region us-west-2`) | `['--env', 'prod', '--region', 'us-west-2']` | prod + region config |
| explicit `--`  | 0               | empty (node + entry only) | defaults       |
| explicit `--`  | 1 (`-- --env prod`) | `['--env', 'prod']` (-- stripped) | prod config |
| explicit `--`  | 2+ (`-- --env prod --region us-west-2`) | `['--env', 'prod', '--region', 'us-west-2']` | prod + region config |

**gap analysis:** none — all combinations covered. separator presence is transparent to wish file.

## matrix.4: arg namespace edge cases

| ind: arg type | ind: conflicts with declastruct | dep: process.argv | dep: declastruct behavior |
|---------------|--------------------------------|-------------------|--------------------------|
| custom flag   | no (`--env prod`)              | contains arg      | passthrough              |
| custom flag   | yes, after `--` (`-- --wish x`)| contains arg      | passthrough (user's namespace) |
| boolean       | no (`--verbose`)               | contains flag     | passthrough              |
| boolean       | no (`--debug --trace`)         | contains flags    | passthrough              |

**gap analysis:** none — after `--`, user owns the namespace completely.
