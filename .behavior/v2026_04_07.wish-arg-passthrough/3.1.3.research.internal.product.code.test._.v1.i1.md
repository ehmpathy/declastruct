## pattern.3: test utilities

**location:** `src/contract/cli/plan.integration.test.ts` (also in apply tests)

**current behavior:**
- `genTempDir()` creates isolated temp directories
- `cloneFixtureWithExid()` clones fixtures with unique IDs
- `createIsolatedWishFile()` generates wish files inline

**citations:**

[5] genTempDir utility
```typescript
const genTempDir = async (): Promise<string> => {
  const testUuid = getUuid();
  const tempDir = resolve(process.cwd(), `src/.test/assets/.temp/${testUuid}`);
  await mkdir(tempDir, { recursive: true });
  return tempDir;
};
```

[6] cloneFixtureWithExid utility
```typescript
const cloneFixtureWithExid = async (input: {
  fixturePath: string;
  tempDir: string;
  exid: string;
}): Promise<string> => {
  const content = await readFile(input.fixturePath, 'utf-8');
  const replaced = content.replace(/__TEST_EXID__/g, input.exid);
  const targetPath = resolve(input.tempDir, basename(input.fixturePath));
  await writeFile(targetPath, replaced, 'utf-8');
  return targetPath;
};
```

**relation to wish:**
- [REUSE] genTempDir and cloneFixtureWithExid for arg passthrough tests
