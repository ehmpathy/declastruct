**location:** `src/contract/cli/invoke.ts`

uses `commander` for CLI parsing. plan command currently accepts:
- `--wish <file>` (required)
- `--into <file>` (required)

**citation [1]:**
```ts
program
  .command('plan')
  .description('Generate a change plan from a wish file')
  .requiredOption('--wish <file>', 'Path to wish file')
```

- `desiredState` is the resource itself (or null if marked for deletion)

**citation [4]:**

**citation [5]:**

**location:** `src/domain.operations/plan/computeChange.ts`

**citation [7]:**
