            # handoff: declastruct --observe flag

            ## .wish

            declastruct should output the full remote state it observed, separate from the diff-based plan.

            ## .why

            when you debug or audit, you need to see:
            - what the remote system actually returned (all fields)
            - which fields were excluded from diff (readonly)
            - the raw state before declastruct's comparison logic

            currently, `plan.json` only contains the diff decisions. the full remote state is lost.

            ## .proposal

            ```sh
            # current
            npx declastruct plan --wish resources.ts --into plan.json

            # proposed: add --observe flag
            npx declastruct plan --wish resources.ts --into plan.json --observe observe.json
            ```

            ## .output format

            `remote.json` would contain the full observed state for each resource:

            ```json
            {
              "observed": [
                {
                  "resourceClass": "DeclaredCloudflareDomainZone",
                  "uniqueKey": { "name": "example.com" },
                  "remote": {
                    "id": "abc123",
                    "name": "example.com",
                    "type": "full",
                    "paused": false,
                    "status": "active",
                    "nameServers": ["gannon.ns.cloudflare.com", "tia.ns.cloudflare.com"],
                    "originalNameservers": ["ns1.oldregistrar.com", "ns2.oldregistrar.com"],
                    "originalRegistrar": "squarespace domains llc (id: 3827)",
                    "createdOn": "2026-04-07T12:50:59.420278Z",
                    "activatedOn": "2026-04-07T14:30:00.000000Z"
                  },
                  "desired": {
                    "name": "example.com",
                    "type": "full",
                    "paused": false
                  },
                  "readonly": [
                    "status",
                    "nameServers",
                    "originalNameservers",
                    "originalRegistrar",
                    "createdOn",
                    "activatedOn"
                  ]
                }
              ],
              "observedAt": "2026-04-07T15:00:00.000Z"
            }
            ```

            ## .usecases

            | usecase | how --observe.remote helps |
            |---------|----------------------------|
            | debug readonly | see which fields were excluded and their values |
            | audit | snapshot of remote state at plan time |
            | drift detection | compare remote.json over time |
            | onboard | understand what cloudflare returns |

            ## .implementation notes

            declastruct already fetches the remote state to compute the plan. this flag would serialize it before the diff logic runs.
