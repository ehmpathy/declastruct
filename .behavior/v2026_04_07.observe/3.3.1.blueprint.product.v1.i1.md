# blueprint: declastruct --snap

## summary

add `--snap <file>` flag to `declastruct plan` command that outputs full remote and wished state before `omitReadonly` is applied.

---

```
src/
├── contract/cli/
│   ├── [~] invoke.ts              # add --snap option to plan command
│   └── [~] plan.ts                # handle snapFilePath, write snapshot
│
├── domain.objects/
│   └── [+] DeclastructSnapshot.ts # new domain object for snapshot structure

    └── [~] planChanges.ts         # return snapshot entries alongside plan

│   └── [+] write snapshot.json   # if --snap provided
│
└── [~] planChanges.ts
    ├── [○] iterate resources
    ├── [○] fetch remoteState
    ├── [○] compute desiredState
    ├── [+] collect snapshot entry  # BEFORE omitReadonly
    │   ├── [←] serialize(remoteState)   # may be null if new
    │   └── [←] serialize(resource)      # always the declared resource
    ├── [○] computeChange (applies omitReadonly)
    └── [+] return { plan, snapshot }
```

## domain objects

### DeclastructSnapshot [+]

```ts
interface DeclastructSnapshot {
  observedAt: IsoTimestamp;
  remote: DeclastructSnapshotEntry[];
  wished: DeclastructSnapshotEntry[];
}

interface DeclastructSnapshotEntry {
  forResource: {
    class: string;
    slug: string;
  };
  state: Record<string, any> | null;  // serialized with _dobj
}
```

## implementation sequence

1. **create DeclastructSnapshot domain object**
   - define interface
   - define DeclastructSnapshotEntry interface

2. **update planChanges to collect snapshot entries**
   - capture remoteState and resource BEFORE omitReadonly
   - serialize with `serialize()` from domain-objects
   - note: use resource (not desiredState) for wished[] — desiredState is null for del() resources
   - return `{ plan, snapshot }` tuple

3. **update plan.ts to handle snapshot**
   - accept `snapFilePath` parameter
   - write snapshot to file if path provided

4. **update invoke.ts to add --snap flag**
   - add `.option('--snap <file>', ...)`
   - pass to executePlanCommand

5. **add integration tests**
   - verify snapshot created/not created
   - verify snapshot structure and content
