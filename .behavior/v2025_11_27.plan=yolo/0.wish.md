enable declapract apply to be run with --plan yolo, where
 if = yolo, we dont validate anything, and simply apply
immediately


why?

so that we can oneline applys for safe scenarios


e.g.,

```ts
// @/.agent/repo=.this/skills/use.vpc.ts
export const getResources = () => {
  const tunnel = DeclaredAwsVpcTunnel.as({
    via: {
      mechanism: 'aws.ssm',
      bastion: { exid: 'vpc-main-bastion' },
    },
    into: {
      cluster: { name: 'ahbodedb' },
    },
    from: {
      host: 'localhost',
      port: 1_5432,
    },
    status: 'OPEN',
  });
  const hostAlias = DeclaredUnixHostAlias.as({
    via: '/etc/hosts',
    from: `aws.ssmproxy.ahbodedb.${env.access}`,
    into: '127.0.0.1',
  });
  return [tunnel, hostAlias]
}
```

should be able to

```sh
npx declapract apply --wish .agent/repo=.this/skills/use.vpc.ts --plan yolo
```

and have it apply whatever plan was passed in, based on the wish directly
